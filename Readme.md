# Nombre del proyecto
GamingWeb es una plataforma web integral diseñada para entusiastas de los videojuegos. El proyecto combina un blog de noticias de última generación con una tienda virtual (Store) y un centro de monitoreo de ciberseguridad, todo bajo una estética moderna "Dark & Neon".

## Descripción
Este proyecto fue creado para centralizar la información más relevante del mundo gaming en 2026. Resuelve la necesidad de tener en un solo lugar:
- Noticias actualizadas de hardware y lanzamientos.
- Una vitrina de productos con contenido multimedia (Video/Trailers).
- Información crítica sobre seguridad digital para jugadores.

## Tecnologías Utilizadas
  - HTML5
  - CSS3

## Requisitos Previos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Editor de código (recomendamos VS Code)
- Extensión Live server

## Instalación
Descarga el archivo .zip y extrae la carpeta 

## Uso

inicia el index.html en el navegador de tu elección
*nota* Para ver el mapa de seguridad y el contenido multimedea correctamente, se recomienda tener instalado una extensión Live Server

## Características

- Navegación Intuitiva: Menú principal consistente en todas las páginas.
- Indice de Contenidos: Sistema de anclas internas en la sección de Noticias para navegación rápida.
- Tienda Multimedia: Uso de etiquetas video> y iframe> para trailers de lanzamientos como GTA VI.
- Seguridad en Tiempo Real: Integración del mapa de ciberataques de Kaspersky.
- Diseño Responsivo: Uso de flex-wrap y aspect-ratio para que la web se adapte a móviles y PCs.
